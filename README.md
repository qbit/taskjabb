taskjabb
========

taskwarrior to jabber interface

requirements
============

[Node.js](http://nodejs.org) must be installed as well as [task](http://taskwarrior.org/projects/show/taskwarrior).

installing
==========


1. git clone https://github.com/qbit/taskjabb.git
2. cd jaskjabb
3. npm install
4. node taskjabb -h
